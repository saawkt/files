HOW TO USE SNAPPY DRIVER INSTALLER ORIGIN

1) Upon launch, click DOWNLOAD INDEXES ONLY, then wait.
2) Select the drivers you wish to install.

REQUIRED DRIVERS:
Audio (required for most microphones)
iGPU (required for most laptops, optional for desktops)

RECOMMENDED DRIVERS:
Chipset (AMD)
GPIO (for laptops)
I2C (for laptops)
NVME (if available)
USB (if available)

OPTIONAL DRIVERS (TEST FOR BEST PERFORMANCE):
SATA
Chipset (Intel)
Bluetooth (if needed)
Host Bridge/DRAM Registers
PCI Express Root Port
PCIe Controller


3) Click the Install button on the left side.
